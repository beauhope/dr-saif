#  الملف الأكاديمي والمهني للدكتور سيف محمد الجابري

هذا المشروع عبارة عن موقع إلكتروني (Portfolio) يعرض السيرة الذاتية الأكاديمية والمهنية للدكتور **سيف محمد الجابري**، مع صفحات تفصيلية تشمل: المؤهلات العلمية، الأوسمة والتقديرات، الكتب والمنشورات، نموذج الشيخ زايد الاستراتيجي، المقالات الفكرية والثقافية، والمسيرة المهنية التخصصية.

##  محتويات المشروع

- `index.html` : الصفحة الرئيسية
- `style.css` : ملف التصميم الموحد (CSS)
- `articles.html` : صفحة المقالات الفكرية والثقافية الرسمية
- `books.html` : صفحة الكتب والمنشورات
- `awards.html` : صفحة الأوسمة والميداليات والتقديرات
- `qualifications.html` : صفحة المؤهلات العلمية
- `specialization.html` : صفحة المسيرة المهنية التخصصية
- `zayed-model.html` : صفحة نموذج الشيخ زايد الاستراتيجي
- `images/` : مجلد الصور والأيقونات المستخدمة في الموقع

---

##  طريقة التشغيل محليًا (على الجهاز)

1. قم بتحميل ملفات المشروع.
2. افتح ملف `index.html` مباشرة عبر المتصفح (Chrome / Edge / Firefox).
3. تأكد أن مجلد `images` موجود في نفس مستوى ملفات الصفحات.

> إذا لم تظهر الصور: تحقق من أسماء الصور ومساراتها داخل ملفات HTML.

---

##  طريقة النشر على GitHub Pages (مختصرة)

1. افتح المستودع في GitHub.
2. اذهب إلى: **Settings → Pages**
3. اختر:
   - **Branch**: `main`
   - **Folder**: `/root`
4. اضغط **Save**
5. سيظهر رابط النشر بعد لحظات.

---

##  التعديل والتطوير

لتعديل محتوى أي صفحة:
- افتح الملف المطلوب (مثل `books.html` أو `specialization.html`)
- عدّل النص ثم احفظ الملف
- قم بعمل **Commit + Push** عبر GitHub Desktop أو عبر GitHub مباشرة
